# 📋 مشخصات فنی ویژگی‌های اولویت اول (MVP)

## FEAT-01: Auth & Profile
- **ورودی:** شماره تلفن $\rightarrow$ کد OTP.
- **خروجی:** دسترسی به داشبورد.
- **جزئیات:** ایجاد نام کاربری @username در اولین ورود.

## FEAT-02: Messaging Core
- **تکنولوژی:** WebSockets (Socket.io).
- **قابلیت‌ها:** ارسال لحظه‌ای، تیک‌های خوانده شدن، Reply و Forward.
- **پین:** ذخیره ID پیام پین شده در دیتابیس برای هر گفتگو.

## FEAT-03: Media Handling
- **ذخیره‌سازی:** استفاده از Object Storage برای فایل‌ها.
- **بهینه‌سازی:** تولید Thumbnail برای عکس‌ها و ویدیوها.

## FEAT-04: PWA & Connectivity
- **Service Worker:** کش کردن فایل‌های استاتیک.
- **Connectivity:** پیاده‌سازی مکانیزم Retry برای محیط‌های با اینترنت ناپایدار.

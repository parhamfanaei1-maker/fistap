# 🎯 تعریف دقیق MVP (اولویت اول)

**موارد اجباری:**
- ورود OTP، پروفایل کامل، @username.
- چت تک‌به-تک، گروه، کانال.
- ارسال رسانه (عکس، ویدیو، ویس، فایل).
- همگام‌سازی مخاطبین.
- PWA و Push Notifications.
- دسترسی بدون VPN.
- تنظیمات پایه حریم خصوصی و تم (Light/Dark).

**موارد حذف شده از MVP (انتقال به اولویت ۲ و ۳):**
- تماس صوتی/تصویری، ربات‌ها، Mini Apps، چت‌های محرمانه، جستجوی پیشرفته.

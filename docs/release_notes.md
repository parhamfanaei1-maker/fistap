# 📦 Release Notes — Fistap v0.1.0 (MVP)

**تاریخ:** 2026-06-12 · **وضعیت:** ✅ عبور از گیت ۳ پروتکل AI-DOS Omnipotent — آماده استقرار

## ✨ قابلیت‌ها (همه الزامات `mvp_definition.md`)

### احراز هویت و پروفایل
- ورود با شماره موبایل + کد OTP پیامکی (نرمال‌سازی کامل فرمت‌های ایرانی و ارقام فارسی)
- سشن امن JWT (access ۱۵ دقیقه + refresh ۳۰ روزه با چرخش ضد سرقت)
- پروفایل با @username یکتا و نام نمایشی

### پیام‌رسانی
- چت تک‌به‌تک، **گروه** و **کانال** (کانال: فقط ادمین‌ها پست می‌کنند) — لحظه‌ای با Socket.io
- تیک‌های وضعیت: ارسال ✓ / تحویل ✓✓ / خوانده‌شده (آبی) — تحویل <۱ ثانیه
- نمایش «در حال نوشتن...»، آنلاین/آخرین بازدید
- **پاسخ (Reply)**، **بازارسال (Forward)** و **سنجاق پیام** با همگام‌سازی لحظه‌ای
- مدیریت گروه: نقش‌های مالک/ادمین/عضو، افزودن/حذف عضو، انتصاب ادمین، تغییر نام

### رسانه
- ارسال عکس، ویدیو، ویس و فایل تا ۵۰MB — آپلود مستقیم به Object Storage (presigned، بدون فشار روی سرور) + نوار پیشرفت

### مخاطبین و جستجو
- همگام‌سازی مخاطبین (Contact Picker اندروید + ورود دستی)، تفکیک «در فیستپ / دعوت»، شروع چت با یک کلیک
- جستجوی جهانی @username/نام/گروه‌ها با debounce

### PWA
- نصب روی صفحه اصلی (اندروید + راهنمای iOS) — manifest کامل RTL/فارسی
- حالت آفلاین: صفحه اختصاصی + کش گفتگوها (NetworkFirst) + بنر وضعیت اتصال
- **اعلان push پس‌زمینه** (Web Push/VAPID) فقط وقتی آفلاین هستید — کلیک = باز شدن چت
- تم روشن/تاریک

## 🔐 امنیت (ممیزی لایه ۴ — صفر یافته بحرانی)
هش OTP + قفل تلاش · چرخش refresh ضد replay · JWT الزامی روی سوکت · ماتریس کامل مجوزها · لیست سفید MIME (ضد XSS) · ضد path-traversal و ReDoS · redact لاگ‌ها · rate-limit لایه‌ای · عدم نشت شماره تلفن در هیچ API عمومی

# ⚙️ قابلیت‌های صفحات (Page Capabilities)

## ۱. صفحه داشبورد (Dashboard)
- **Sidebar:** 
    - فیلتر کردن چت‌ها.
    - جابجایی بین گفتگوها.
    - دسترسی سریع به تنظیمات.
- **Chat Window:**
    - ارسال/دریافت پیام در لحظه.
    - ارسال فایل/عکس/ویس.
    - مشاهده پیام‌های پین شده.
    - نمایش وضعیت آنلاین/تایپینگ.

## ۲. صفحه مخاطبین (Contacts)
- **Sync:** فراخوانی Contact Picker API.
- **Matching:** تطبیق شماره‌ها با دیتابیس.
- **Invite:** تولید لینک دعوت برای مخاطبان غیرعضو.

## ۳. صفحه تنظیمات (Settings)
- **Profile Edit:** تغییر نام و عکس.
- **Privacy Control:** مدیریت نمایش اطلاعات شخصی.
- **Notification Toggle:** فعال/غیرفعال کردن Push Notifications.

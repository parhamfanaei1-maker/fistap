# 🗺️ نقشه صفحات پروژه Fistap (Page Map)

## ۱. Auth Flow (جریان ورود)
- `/auth/welcome`: صفحه ورود شماره تلفن.
- `/auth/verify`: صفحه تایید کد OTP.
- `/auth/setup`: صفحه تکمیل پروفایل اولیه.

## ۲. Main Layout (ساختار اصلی)
- `/app/dashboard`: صفحه اصلی (شامل Sidebar و Chat Window).
- `/app/contacts`: صفحه مدیریت و همگام‌سازی مخاطبین.
- `/app/search`: صفحه جستجوی جهانی کاربران و کانال‌ها.

## ۳. Management (مدیریت)
- `/app/settings`: صفحه تنظیمات کلی و حریم خصوصی.
- `/app/group-settings/:id`: صفحه مدیریت تنظیمات گروه/کانال.

## ۴. Profile (پروفایل)
- `/app/profile/:id`: مشاهده پروفایل کاربر دیگر یا خود کاربر.

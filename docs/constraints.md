# ⚠️ محدودیت‌ها و پیش‌نیازها (Constraints)

## ۱. محدودیت‌های فنی
- **پلتفرم:** سیستم باید به صورت Web-based باشد و استانداردهای PWA را رعایت کند.
- **دسترسی:** عدم نیاز به VPN برای کاربران ایرانی (الزام به انتخاب زیرساخت مناسب).
- **اعتبارسنجی:** وابستگی به سرویس‌های ارسال پیامک برای OTP.

## ۲. محدودیت‌های عملکردی
- **مقیاس:** شروع با ۱,۰۰۰ کاربر (Baseline).
- **پایداری:** زمان فعال بودن (Uptime) باید بالای ۹۹.۹٪ باشد.

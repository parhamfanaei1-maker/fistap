## 📐 Grid System

```
دسکتاپ (≥769px): Container max-width 1440px
12-column grid, gutter: 24px

Sidebar: 3 cols (280px fixed)
Chat List: 4 cols (360px fixed)
Chat Window: 5 cols (flexible)

موبایل (≤768px): 100vw, Padding 16px, Single column
```

## 📐 Spacing Scale

| Token | Value | کاربرد |
|---|---|---|
| `space-xs` | 4px | فاصله آیکون تا متن |
| `space-sm` | 8px | Padding داخلی |
| `space-md` | 16px | فاصله بین المان‌ها |
| `space-lg` | 24px | فاصله بین بخش‌ها |
| `space-xl` | 32px | فاصله بین بخش‌های اصلی |
| `space-2xl` | 48px | فاصله Header/Content |

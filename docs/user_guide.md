# 📖 راهنمای کاربر فیستپ (User Guide) — v0.1.0

## شروع سریع

### ۱. ورود به فیستپ
1. آدرس فیستپ را در مرورگر باز کنید (نیازی به VPN نیست).
2. **شماره موبایل** خود را وارد کنید (مثلاً `09121234567`) — ارقام فارسی هم پذیرفته می‌شود.
3. **کد ۵ رقمی** پیامک‌شده را وارد کنید (کد خودکار از پیامک خوانده می‌شود در اندروید). تایمر ۲ دقیقه‌ای دارید؛ پس از آن «ارسال مجدد» فعال می‌شود.
4. بار اول: **نام نمایشی** و **نام کاربری** (`@username`) بسازید — نام کاربری با حرف انگلیسی شروع شود، ۵ تا ۳۲ کاراکتر.

### ۲. نصب روی گوشی (PWA)
- **اندروید (Chrome):** بنر «نصب فیستپ» را بزنید یا منوی مرورگر → *Install app*.
- **iOS (Safari):** دکمه اشتراک‌گذاری → **Add to Home Screen**.
- پس از نصب، فیستپ مثل یک اپ مستقل با آیکون خودش باز می‌شود.

## گفتگو

| کار | چگونه |
|---|---|
| شروع چت با دوست | **مخاطبین** → «انتخاب از مخاطبین گوشی» (یا ورود دستی شماره) → دکمه **گفتگو** کنار نام |
| پیدا کردن کاربر/گروه | **جستجو** → تایپ `@username` یا نام |
| ارسال پیام | تایپ + Enter (Shift+Enter = خط جدید) |
| ارسال عکس/ویدیو/ویس/فایل | آیکون 📎 کنار کادر پیام (تا ۵۰MB، با نوار پیشرفت) |
| **پاسخ (Reply)** | دابل‌کلیک روی پیام → نوار «پاسخ به» بالای کادر |
| **بازارسال (Forward)** | hover روی پیام → آیکون ↪ → انتخاب گفتگوی مقصد |
| **سنجاق (Pin)** | hover روی پیام → آیکون 📌 — نوار سنجاق بالای چت برای همه ظاهر می‌شود؛ کلیک = پرش به پیام |

### معنی تیک‌ها
✓ ارسال شد · ✓✓ به گوشی مخاطب رسید · ✓✓ آبی = خوانده شد · 🕓 در حال ارسال · ✕ ناموفق

## گروه و کانال

- **ساخت:** داشبورد → دکمه «+ جدید» (موبایل: دکمه شناور آبی) → نام + کلید «کانال؟»
  - **گروه:** همه اعضا پیام می‌فرستند · **کانال:** فقط مالک و ادمین‌ها
- **مدیریت:** داخل گفتگو → آیکون ⚙ بالا → تغییر نام، افزودن/حذف عضو، کلید «ادمین» کنار هر عضو (فقط مالک)
- **خروج از گروه:** در همان صفحه، آیکون سطل کنار نام خودتان

## تنظیمات
- **اعلان‌ها:** کلید «اعلان پیام‌های جدید» — حتی وقتی فیستپ بسته است اعلان می‌گیرید (کلیک روی اعلان = باز شدن چت)
- **حالت تاریک:** کلید مخصوص · **خروج از حساب:** پایین صفحه

## حالت آفلاین
بدون اینترنت، گفتگوهای اخیر قابل مشاهده‌اند و بنر وضعیت بالای صفحه ظاهر می‌شود؛ با وصل شدن، همه‌چیز خودکار همگام می‌شود.

## پرسش‌های متداول
- **کد پیامک نیامد؟** پس از پایان تایمر، «ارسال مجدد» را بزنید (حداکثر ۵ درخواست در ۱۵ دقیقه).
- **«تلاش بیش از حد»؟** ۵ بار کد اشتباه = کد باطل؛ کد جدید بگیرید.
- **اعلان نمی‌آید؟** تنظیمات مرورگر/سیستم‌عامل → مجوز Notification برای دامنه فیستپ + روشن بودن کلید در تنظیمات اپ.

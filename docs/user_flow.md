## 📊 ماتریس تصمیم‌گیری (Decision Matrix)

| Decision Point | شرط | مسیر Yes | مسیر No |
|---|---|---|---|
| DP-01 | JWT Token معتبر؟ | → Dashboard | → Auth Flow |
| DP-02 | شماره تلفن معتبر؟ | → OTP Verify | → Error + Retry |
| DP-03 | OTP صحیح؟ | → Check Registration | → Error + Retry |
| DP-04 | کاربر ثبت‌نام شده؟ | → Dashboard | → Profile Setup |
| DP-05 | Username یکتا؟ | → Create Session | → Error + Retry |
| DP-06 | دسترسی مخاطبین؟ | → Sync Contacts | → Request Permission |
| DP-07 | اینترنت فعال؟ | → Online Mode | → Offline Queue |
| DP-08 | فایل آپلود شد؟ | → Show in Chat | → Retry / Error |

## 🔄 حالت آفلاین (Offline Mode)

```
[Internet Connection Lost]
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│   نوار وضعیت: "در حال تلاش برای اتصال..."                  │

│   • آپلود فایل‌ها → Pause و Queue                           │

    │
    ▼ (Connection Restored)
[Sync Queue with Server]
    ├── ارسال پیام‌های ذخیره شده
    ├── ادامه آپلود فایل‌ها
    ├── دریافت پیام‌های جدید
    └── به‌روزرسانی وضعیت‌ها
```
